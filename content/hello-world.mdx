### Semua Berawal dari “Hello World” 👋

Halo semua! 🎉
Selamat datang di blog pertama saya yang berjudul “All Start With 'Hello World'”. Tulisan ini saya buat dalam bahasa Indonesia karena saya ingin menyampaikan cerita dan pengalaman saya secara lebih personal dan mengalir. ### Awal Ketertarikan pada Teknologi 💡

Minat saya terhadap teknologi sebenarnya sudah muncul sejak saya duduk di bangku kelas 4 SD. Namun sebelum benar-benar tertarik dengan dunia digital, saya sempat menyukai hal-hal yang berkaitan dengan kelistrikan ⚡. Saya sangat senang bermain alat-alat tukang—mulai dari palu mainan hingga obeng plastik. Bahkan, saya pernah memukul tembok rumah dengan alat mainan hingga temboknya rusak 😅.

Melihat hal tersebut, orang tua saya akhirnya membelikan saya satu set toolbox berisi peralatan asli seperti palu, obeng, tang, bahkan helm proyek. Momen itu cukup membekas dan menjadi awal dari ketertarikan saya pada hal-hal teknis.

Ketertarikan terhadap dunia digital mulai berkembang ketika saya bertanya pada diri sendiri:

> “Bagaimana cara kerja Google sebenarnya?” 🤔

Pertanyaan sederhana tersebut membuat saya mulai mencari tahu secara otodidak melalui internet dan video-video edukatif.

Saat SMP, tepatnya pada masa pandemi, saya mulai mengenal dasar-dasar pemrograman melalui mata pelajaran TIK. Di sana, saya diperkenalkan dengan **HTML** dan dasar-dasar pembuatan website. Dari situlah saya mulai belajar secara mandiri, terutama melalui channel YouTube Web Programming UNPAS milik Pak Sandhika Galih—yang sangat membantu membuka jalan saya menuju dunia pengembangan web.

Selain belajar pemrograman web, saya juga sempat mencoba bidang **robotika** dan membuat game sederhana menggunakan **C++**. Pengalaman tersebut sangat menyenangkan dan memperkuat minat saya dalam dunia teknologi. 🤖
